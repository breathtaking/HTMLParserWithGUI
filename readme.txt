Приложение с графическим интерфейсом на основе JavaFX
Назначение: разбор входящих HTML данных

Возможно использование одного из 3 видов источников:
1)URL веб страницы
2)Файл на компьютере (необходимо указать путь)
3)Строка, содержащая код HTML

Запуск приложения:
1. com.javafxapp.MainApp - точка входа (если открывать проект через IDE).

2. Упакованный .jar файл можно запустить через терминал командой
java -classpath HTMLParserWithGUI.jar com.javafxapp.MainApp

Работа с приложением:
После запуска скопируйте в одно из текстовых полей ссылку на вэб страницу,
путь к файлу на компьютере либо текст с HTML кодом,
после чего необходимо нажать кнопку "Parse" напротив нужного поля.
Результаты разбора HTML кода появятся в правой части окна приложения.

P.S. В некоторых пакетах остались классы, с помощью которых приложение
могло работать в консольном режиме.
